# Installing SASS

On a Mac:
```
npm install -g sass
```
[^1]


[^1]:

# Running SASS Compiler
Go to Directory in CLI
```
sass --watch style.scss:style.css
```

Tutorial: https://www.tutorialspoint.com/sass/sass_installation.htm


# Running SASS Shell

```
sass -i
```
